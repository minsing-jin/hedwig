🦉 Hedwig v3 — Concept Demo

Algorithm Sovereignty for individuals. 기업 알고리즘 대신, 당신이 자연어·👍/👎·질문으로 조각하는 자기진화 추천 엔진.

⚡ 1분 Setup — 시드 데이터 주입

실제 daily 파이프라인 돌리기 전에 synthetic 시그널/피드백/진화 이벤트를 DB에 심어서 timeline, sandbox, meta 페이지를 즉시 의미있게 체험합니다. 실제 사용자 데이터(criteria.yaml, 실제 feedback)는 건드리지 않습니다 — 시드는 external_id demo-* 전용.
